# Admin Dashboard

A modern, pixel-perfect **Admin Dashboard UI** built with React, TypeScript, and Tailwind CSS.  
This project focuses on clean layout architecture, reusable components, and close alignment with Figma designs.

---

## üöÄ Features

### Dashboard
- KPI stat cards
- Clean layout with sidebar, header, and optional right panel

### Orders Management
- Fully responsive orders table
- Search functionality
- Column sorting
- Pagination
- Row selection with indeterminate checkbox state
- Status indicators with color-coded badges
- Text truncation (ellipsis) for long project names
- Calendar icon integration for the date column

### Routing
- Client-side routing using **React Router**
- Dashboard and Orders pages share a common layout

### Design Accuracy
- Built to closely match Figma specifications
- Exact spacing, typography, colors, and component sizing
- Uses design tokens and CSS variables instead of hardcoded values
